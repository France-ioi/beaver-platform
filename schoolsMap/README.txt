
USAGE:

   - run 
     localhost/beaver_platform/schoolsMap/updateSchoolCoordinates.php
     (by default it only completes missing coordinates,
     but still it can be very slow)
   - set CERTIGEN_CONTESTS and YEAR_CONTEST in genSchoolsMap.php
   - open localhost/beaver_platform/schoolsMap/genSchoolsMap.php
   - preview the target file, e.g:
     localhost/beaver_platform/schoolsMap/map.html
   - copy the target file, e.g.
     cp beaver_platform/schoolsMap/map.html beaver/www/map/map2014.html
